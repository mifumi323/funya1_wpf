# funya1_wpf
ふにゃ1のベタ移植プロジェクト。

ゲームエンジンなどは使わず、なるべく当時と同等の仕組みで実現する。

## VB版からの主な変更点
- 対応環境をWindows 7以降に変更
- 設定項目関連をファイルメニューからオプションメニューに移動
- メニューのショートカットキーを追加
- バナナのデザインをオリジナルに変更
- WASDキーによる操作に対応
- ウィンドウタイトルにステージ名だけでなくゲーム名も表記
- メッセージの背景をちょっと透かした
- メッセージ表示中にメニュー操作可能に
- メッセージに閉じるボタンを追加
- メッセージ右下に補足説明を追加
- 秘密機能解放してもゲーム閉じない
- 画面サイズを可変にするオプションを追加
- 音楽をループするかどうかのオプション追加
- 設定の保管場所変更
- ゲームの進行だけでなく、設定も保存するように変更
- プレイ記録の画面を全面リニューアル
- ドラッグ＆ドロップでステージファイルを読み込めるように
- 背景画像読み込み失敗時はサンプルステージの画像で置き換え
- ステージファイル読み込み失敗したときは元のステージのままに
- バージョン情報のシステム情報ボタンを廃止
- バージョン情報のアニメーションパターン追加
- ヘルプの形式変更
- ステージメーカーをゲーム本体に統合
