# funya1_wpf
ふにゃ1のベタ移植プロジェクト。

ゲームエンジンなどは使わず、なるべく当時と同等の仕組みで実現する。

## VB版からの主な変更点
- 対応環境をWindows 7以降に変更
- 設定項目関連をファイルメニューからオプションメニューに移動
- バナナのデザインをオリジナルに変更
- WASDキーによる操作に対応
- ウィンドウタイトルにステージ名だけでなくゲーム名も表記
- メッセージの背景をちょっと透かした
- メッセージ表示中にメニュー操作可能に
- メッセージに閉じるボタンを追加
- 秘密機能解放してもゲーム閉じない
- 画面サイズを可変にするオプションを追加
- 音楽をループするかどうかのオプション追加
- 設定の保管場所変更
